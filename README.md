# DRAIS - School Management & Attendance System

## Website & Marketing Repository

This is the public website repository for DRAIS (Digital Record and Attendance Information System) - a comprehensive school management system designed for East African schools.

### 🚀 Features Showcase

- **Biometric Attendance** - Fingerprint-powered attendance monitoring
- **Student Management** - Comprehensive student information systems  
- **Parent Communication** - Real-time notifications and updates
- **Report Generation** - Professional report cards and analytics
- **Islamic Education Support** - Specialized tools for Quranic studies

### 🌐 Live Demo

Visit our website to learn more about DRAIS and see the system in action:
- Features overview
- Interactive attendance demo
- Real testimonials from East African schools
- Pricing and packages
- Documentation and setup guides

### 🎨 Tech Stack

- **Frontend**: Next.js 14 with TypeScript
- **Styling**: Tailwind CSS with dark/light theme support
- **Icons**: Lucide React
- **Deployment**: Vercel

### 🏃‍♂️ Quick Start

```bash
# Clone the repository
git clone https://github.com/xhenovolt/drais-website.git
cd drais-website

# Install dependencies
npm install

# Run development server
npm run dev
```
